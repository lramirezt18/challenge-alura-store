# 🛍️ Challenge Alura Store - Análisis de Datos

## 📊 Descripción del proyecto

Este proyecto tiene como objetivo analizar el rendimiento de **cuatro tiendas de la cadena Alura Store** para ayudar al **Sr. João** a tomar una decisión estratégica sobre cuál tienda debería vender para iniciar un nuevo emprendimiento.

Para realizar este análisis se utilizaron herramientas de **análisis de datos en Python**, evaluando diferentes indicadores como ingresos, productos vendidos, satisfacción de clientes y costos de envío.

---

## 🎯 Objetivo

Identificar **la tienda con menor rendimiento** para recomendar cuál debería vender el Sr. João basándonos en el análisis de los datos.

---

## 🧰 Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab / Jupyter Notebook

---
## 🔎 Análisis realizados

### 1 Ingreso total por tienda
Se calcularon los ingresos totales sumando los valores de la columna **Precio** para cada tienda.

### 2 Categorías más vendidas
Se analizaron las categorías de productos más populares para identificar tendencias de consumo.

### 3 Valoración promedio de clientes
Se calculó el promedio de la **calificación de los clientes** para evaluar la satisfacción en cada tienda.

### 4 Productos más y menos vendidos
Se identificaron los productos con **mayor y menor número de ventas** en cada tienda.

### 5 Costo de envío promedio
Se analizó el costo promedio de envío para entender su impacto en las ventas.

## 📈 Visualizaciones

Se generaron diferentes tipos de gráficos para facilitar la interpretación de los datos:

- Gráficos de barras
- Gráficos circulares
- Gráficos de líneas

Estas visualizaciones permiten entender mejor los patrones de ventas entre las diferentes tiendas.

---

## 💡 Recomendación final

Después de analizar los datos de las cuatro tiendas, se observa que **la Tienda 4 presenta el menor nivel de ingresos en comparación con las demás**.

Aunque las calificaciones de los clientes y los costos de envío son relativamente similares entre las tiendas, la diferencia en ingresos indica que **la Tienda 4 tiene un rendimiento inferior**.

### 📌 Recomendación

Se recomienda que **el Sr. João considere vender la Tienda 4**, ya que su desempeño es menor frente a las otras tiendas. Esto le permitiría concentrar sus recursos en las tiendas con mayor rendimiento y potencial de crecimiento.
